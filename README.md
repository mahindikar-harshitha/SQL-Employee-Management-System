# Employee Management System (SQL Project)

## Overview

The Employee Management System is a SQL-based database project designed to manage employee information, job roles, departments, salaries, qualifications, leave records, and payroll data within an organization. The project demonstrates relational database design, data integrity through constraints, and business analysis using SQL queries.

## Features

* Employee Information Management
* Department and Job Role Management
* Salary and Bonus Tracking
* Qualification and Skills Management
* Leave and Absence Tracking
* Payroll Processing and Reporting
* HR Analytics and Business Insights

## Database Schema

The project consists of six interconnected tables:

1. JobDepartment
2. Employee
3. SalaryBonus
4. Qualification
5. Leaves
6. Payroll

## SQL Concepts Used

* DDL Commands (CREATE, ALTER)
* Constraints (PRIMARY KEY, FOREIGN KEY, UNIQUE)
* Joins
* Aggregate Functions
* Subqueries
* GROUP BY & ORDER BY
* Cascading Actions
* Data Analysis Queries

## Key Analysis Performed

* Employee and Department Insights
* Salary and Payroll Analysis
* Qualification Tracking
* Leave Pattern Analysis
* Bonus Distribution Analysis

## Tools Used

* MySQL
* MySQL Workbench

## Business Outcome

The system centralizes employee-related data, improves data consistency, streamlines HR operations, and supports data-driven decision-making through SQL-based reporting and analytics.
